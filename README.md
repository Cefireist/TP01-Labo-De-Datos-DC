# TP01 - Laboratorio de Datos  
**Departamento de Computación - FCEN - UBA**

Este trabajo práctico explora la relación entre los **centros culturales** y los **establecimientos educativos** en Argentina, utilizando datos oficiales del Censo 2023.

## Objetivo
Analizar si existe una relacion entre la cantidad de centros culturales y la presencia de establecimientos educativos, considerando también la distribución de la población.

## Metodología

1. **Recolección de datos**:
   - Establecimientos educativos
   - Centros culturales
   - Datos poblacionales
   - Todos provenientes del Censo Nacional 2023 (Argentina)

2. **Procesamiento de datos**:
   - Limpieza, filtrado y transformación de los datasets.
   - Uso de SQL embebido en Python mediante **DuckDB** para consultas eficientes.

3. **Análisis y visualización**:
   - Análisis exploratorio con **pandas**, **numpy** y **matplotlib/seaborn**.
   - Elaboración de un informe con los principales resultados.
